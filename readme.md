### 환경변수 파일 필요합니다.

`.env.development.local`
```
REACT_APP_API_URL=http://localhost:3001
REACT_APP_REDIRECT_URL=https://localhost:3000
```

### 폴더 루트에 cert.pem, key.pem 필요합니다.

### 실행
```
npm i
npm run server
npm start
```
